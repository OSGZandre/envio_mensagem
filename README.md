# Projeto Symfony - Design Patterns

## Sobre

Projeto simples desenvolvido para estudo de Design Patterns, com foco em Abstract Factory e Decorator.

## Implementação

* Abstract Factory: responsável pela criação de diferentes tipos de mensagens (ex: Email e SMS), desacoplando a lógica de criação do uso.
* Decorator: utilizado para adicionar comportamentos extras às mensagens sem modificar as classes originais.

## Tecnologias

* PHP
* Symfony
* Composer

## Execução
* É necessário a instalação do PHP, Symfony, e composer. 
```bash
symfony check:requirements
```
```bash
symfony serve
```

Acesse:

```
http://127.0.0.1:8000
```

## Objetivo

Aplicar de forma prática e simples os conceitos dos padrões Abstract Factory e Decorator.
